## Описание программы 

**Task Manager** - Это простое графическое приложение для учёта личных или рабочих дел. Оно позволяет создавать список задач, управлять ими и сохранять данные на диск.

### Возможности:
- Добавление своих своих задач
- Выдача задачам их приоретет, то есть нужность выполнения 
- Фильтрация поставленных задач 

### Инстркция по запуску 
- Python версии 3.6 или выше 
- Установленная библеотка Tkinter (Обычно включен, как стандартная библеотека с 3.1 )

### Запуск 
Скопируйте код в файл task_manager.py и запустите его командой: python task_manager.py

### Примеры использования 
**Заполните заголовок:** "Сходить в магазин"
**заполните описание:** "Купить хлеб, молоко, яйца"
**Выберите приоритет:** "Срединй, низкий или высокий"
